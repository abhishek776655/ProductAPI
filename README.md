# ProductAPI

## How to run?
     To run server 
     run the following commands

     1. git clone https://github.com/abhishek776655/ProductAPI.git
     2. cd ProductAPI
     3. npm start

## How to test?
     To run tests just enter the directory execute command

     npm run test
